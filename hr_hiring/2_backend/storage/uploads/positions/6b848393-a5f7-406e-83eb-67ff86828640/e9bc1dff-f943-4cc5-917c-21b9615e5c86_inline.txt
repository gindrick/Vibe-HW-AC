Senior Frontend Developer
Hledáme zkušeného parťáka/parťačku, který/á nám pomůže posunout naše webové aplikace na další úroveň. Pokud tě baví čistý kód, moderní technologie a chceš mít reálný vliv na architekturu produktu, jsi náš člověk.

Co bude tvým úkolem?
Vývoj a architektura: Návrh a implementace komplexních frontendových aplikací.

Kód pod kontrolou: Psaní čistého, testovatelného a udržitelného kódu.

Mentoring: Sdílení know-how s týmem a provádění code reviews.

Inovace: Sledování moderních trendů a jejich rozumné nasazování do praxe.

Spolupráce: Úzká kooperace s designéry (UX/UI) a backendisty na návrhu API.

Co od tebe očekáváme?
Seniorní zkušenost: Minimálně 5 let v profesionálním vývoji frontendu.

Technologický stack: Špičková znalost Reactu (případně Vue/Angular), TypeScriptu a moderního CSS.

Architektonický nadhled: Schopnost navrhnout strukturu aplikace tak, aby se v ní nikdo neztratil.

Nástroje: Zkušenost se stavěním CI/CD pipeline, testováním (Jest, Cypress) a verzováním (Git).

Mindset: Proaktivní přístup, smysl pro detail a schopnost dotahovat věci do konce.
Technologický Stack
Frontend (Uživatelské rozhraní)
Framework: React.js nebo Next.js (pro skvělé SEO a rychlost).

Stylování: Tailwind CSS (rychlý vývoj, moderní design) nebo Styled Components.

State Management: Redux Toolkit nebo React Context API.

Backend (Serverová část)
Runtime: Node.js s frameworkem Express nebo NestJS.

API: REST API pro klasiku, nebo GraphQL, pokud potřebuješ tahat komplexní data efektivněji.

Autentizace: JWT (JSON Web Tokens) nebo integrace přes NextAuth/Clerk.

Databáze a Storage
Relační databáze: PostgreSQL (ideální pro strukturovaná data a složité vztahy).

NoSQL: MongoDB (pokud preferuješ flexibilní JSON-like strukturu).

Caching: Redis pro bleskové načítání často používaných dat.

Infrastruktura a Deployment
Hosting: Vercel (pro frontend) a AWS nebo DigitalOcean (pro backend).

CI/CD: GitHub Actions pro automatické testování a nasazování kódu.

Kontejnerizace: Docker, aby vše běželo stejně u tebe v počítači i na serveru.
